# Sample Workspace

This directory provides a tiny, safe workspace for your first AegisLoop run.

It intentionally contains:

- No API keys or secrets
- No external services
- No build tools
- No dependencies
- Small editable files

## Files

| File      | Purpose                             |
| --------- | ----------------------------------- |
| notes.txt | Simple text for reading and editing |
| todo.md   | Small Markdown checklist            |

## Try it with AegisLoop

Open this folder as your workspace and ask ChatGPT to generate a Codex task.

Example prompts:

- Read `notes.txt` and summarize it.
- Fix grammar in `notes.txt`.
- Add one new item to `todo.md`.
- Convert the checklist into numbered steps.

These changes are completely local and safe to experiment with.

When you're finished, simply delete this folder or restore the files.
