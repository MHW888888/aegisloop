# Briefing Templates

Copy these files into a Run Capsule `inbox\` directory, then edit the copied files for one runner thread.

Recommended layout:

```text
C:\AegisLoopRuntime\
  runs\
    PROJECT_ID\
      ACTIVE_BRANCH\
        RUN_ID\
          capsule.json
          inbox\
            GPT_THREAD_BRIEF.md
            CODEX_EXECUTION_BRIEF.md
            RESEARCH_RULES.md
            FROZEN_BRANCHES.md
            CURRENT_OBJECTIVE.md
```

Use `GPT_THREAD_BRIEF.md` in the ChatGPT page.

Keep the other files local for Codex to read after dispatch.
