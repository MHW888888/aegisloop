# AegisLoop Growth Checklist

Use this as a lightweight launch plan. The goal is not spam. The goal is to make the project easy to understand, easy to trust, and easy to share.

## Already Done

- Public GitHub repository
- MIT license
- README with quick start and safety model
- Architecture diagram
- GitHub Actions syntax check
- Release `v0.1.0`
- Topics for discoverability
- Launch post drafts
- Issue and PR templates
- Contributing guide

## Next High-Impact Steps

1. Pin the repository on the GitHub profile.
2. Add one short GIF or screenshot showing the loop panel.
3. Post the short launch note on X, LinkedIn, V2EX, Hacker News "Show HN", Reddit, or a developer WeChat group.
4. Ask 3-5 technical friends to read the README and tell you where they get confused.
5. Turn the clearest confusion into a README improvement.
6. Open 2-3 "good first issue" items for small community tasks.

## Suggested One-Liner

AegisLoop is a guarded local bridge that lets ChatGPT plan and Codex execute, with local gates, workspace locks, dedupe, and audit logs.
